# Avro file format
Experimentations with Avro.


### Prerequisites
+ libsnappy-dev
+ [fastavro](https://pypi.org/project/fastavro/)
+ [python-snappy](https://pypi.org/project/python-snappy/)

Python libs:
``` bash
sudo apt-get install libsnappy-dev
pip install fastavro
pip install python-snappy
```
